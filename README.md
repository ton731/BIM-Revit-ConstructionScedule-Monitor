# Capstone_Monitor
- 用Revit API與BIM模型監測施工各個階段的變位、流程
- --------------------------

- 利用GUI上不同階段的按鍵，可看出在不同階段時，施工的進度以及變位量。
- 協助專業或是非相關專業的人士，都能夠透過此工具，快速的評估以及了解施工各個階段時，施工真實狀況與理想分析狀況之差異、危險處等等
